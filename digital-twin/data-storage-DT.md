# Data Storage in Digital Twin

Data storage is to store the collected data for further processing, analysis, and management.

- Traditional database:
	- mySQL
- Big data storage:
	- distributed file storage (DFS),
	- NoSQL database,
	- NewSQL database,
	- cloud storage
